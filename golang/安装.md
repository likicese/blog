# 安装

## 安装

``` bash
# 安装golang
cd /usr/local
wget https://studygolang.com/dl/golang/go1.15.linux-amd64.tar.gz
tar -xf go1.15.linux-amd64.tar.gz

# 设置go get 代理
export GOPROXY=https://goproxy.io
```

## 设置环境变量

在 /etc/profile 中添加

``` txt
export GOROOT=/usr/local/go
export PATH=$PATH:$GOROOT/bin
```
