## 配置环境

### windows

在系统环境中增加一个环境变量：JAVA_HOME=D:\tool\openjdk

修改一个环境变量：Path 增加：%JAVA_HOME%\bin

### linux

``` bash
yum install java-1.8.0-openjdk-devel
yum install java-1.8.0-openjdk
```
