## 初始化工程

``` bash
mkdir projectName
cd projectName
go mod init aaa  # 建立 go.mod 文件，执行build命令时，会产生名为aaa的模块
```

创建`main.go`文件，内容如下

``` go
package main

import (
        "fmt"
        "github.com/satori/go.uuid"
)

func main() {
        uid := uuid.NewV4();
        fmt.Println(uid)
        fmt.Println("hello world")
}
```

执行`go build`，编译文件
