# 报错解决

## 指令过长

错误：

```
you should increase server_names_hash_bucket_size: 32
```

解决：

```
# 在nginx的配置文件中添加如下一行
http {
    server_names_hash_bucket_size  64;
}
```

## 自编译时，未添加ssl模块

错误：

```
nginx: [emerg] the "ssl" parameter requires ngx_http_ssl_module
```

解决：

``` bash
./configure --with-http_ssl_module  # 重新编译，以添加该模块
```
